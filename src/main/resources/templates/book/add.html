Thêm Sách Mới

Điền thông tin chi tiết để thêm sách vào hệ thống hoặc import từ file Excel

Các trường có dấu * là bắt buộc
Thông Tin Sách
Định dạng: JPG, PNG, GIF. Kích thước tối đa: 5MB
Quay Lại
Hướng dẫn: File Excel phải có định dạng như sau:
  • Cột A (Title): Tên sách
  • Cột B (Author): Tác giả
  • Cột C (Price): Giá (số)
  • Cột D (Category): Tên danh mục (phải trùng với danh mục trong hệ thống)
  • Cột E (Image URL): Đường dẫn ảnh (tùy chọn)
Dòng đầu tiên là tiêu đề cột và sẽ bị bỏ qua
Chọn File Excel
Định dạng: .xlsx hoặc .xls
Quay Lại
Tải file Excel mẫu:
Tải xuống file Excel mẫu
File mẫu đã có dữ liệu ví dụ, bạn có thể chỉnh sửa và import vào hệ thống
Hướng dẫn:
  1. Tải file Excel mẫu ở trên
  2. Chỉnh sửa hoặc thêm dữ liệu sách của bạn
  3. Đảm bảo tên danh mục (Category) phải trùng khớp với danh mục trong hệ thống
  4. Lưu file và upload vào form bên trên
